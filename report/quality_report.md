# Отчёт о качестве данных

## 1. Введение

**Датасет:** Sleep and Lifestyle Health DataSet 

**Источник:** [Sleep and Lifestyle Health DataSet  ](https://www.kaggle.com/datasets/shraddha4ever20/sleep-and-lifestyle-health-dataset) 

**Размер:** 1000 строк × 15 столбцов 

**Предметная область:** Взаимосвязь между поведенческими привычками (курение, потребление кофеина и алкоголя, физическая активность) и метриками качества сна у взрослых людей в возрасте 18–80 лет. 

**Условный заказчик:** Медицинский исследовательский центр, изучающий влияние образа жизни (курение, употребление алкоголя, физическая активность) на качество сна и риск нарушений (низкая эффективность, преждевременные пробуждения).  

**Возможные задачи ИАД:**
1. Выявление ключевых факторов образа жизни, ассоциированных с низкой эффективностью сна
2. Понять, как кофе, алкоголь, сигареты и спорт меняют структуру сна — например, уменьшают ли они долю глубокого сна, который важен для восстановления организма.
3. Оценка влияния количества ночных пробуждений ('Awakenings') на общее качество и продолжительность сна

## 2. Паспорт датасета

| Столбец | Тип | Описание | Примечания |
|---------|-----|----------|------------|
| ID  | int64 | Уникальный ID | Не используется в анализе; можно удалить |
| Age  | int64 | Возраст человека | Диапазон 18–80 лет; выбросов нет |
| Gender  | object | Пол человека | Бинарный: 'Male' / 'Female'; сбалансирован (~50/50) |
| Bedtime  | object | Время, когда человек ложится спать | Хранится как строка в формате YYYY-MM-DD HH:MM:SS; требует преобразования в числовой формат |
| WakeupTime  | object | Время пробуждения человека | Аналогично Bedtime — нуждается в парсинге |
| SleepDuration  | float64 | Общее количество часов сна между временем отхода ко сну и временем пробуждения. | В данных встречаются значения < 4 ч (требует проверки на корректность) |
| SleepEfficiency  | float64 | Процент времени, проведенного во сне; рассчитывается как (продолжительность сна / время, проведенное в постели) × 100. | Значения в диапазоне 0.0–1.0 |
| REMSleepPercentage  | int64 | Процент времени, проведенного в фазе быстрого движения глаз (REM-фаза) во время сна, связанный со сновидениями и памятью. | Целочисленные значения; сумма с другими фазами должна быть ≈100% |
| DeepSleepPercentage  | int64 | Процент времени, проведенного в фазе глубокого сна (медленноволнового сна), которая имеет решающее значение для физического восстановления и иммунитета. | Часть общей структуры сна |
| LightSleepPercentage  | int64 | Процент времени, проведенного в фазе легкого сна, которая составляет большую часть цикла сна | Иногда сумма трёх фаз ≠ 100% — требуется нормализация |
| Awakenings  | int64 | Количество пробуждений за ночь | Целые значения от 0 до 5; дискретный признак |
| CaffeineConsumption  | float64 | Потребление кофеина (мг/день) | 256 пропусков (25,6%); принимает только значения 0.0, 25.0, 50.0 — дискретный признак |
| AlcoholConsumption  | int64 | Употребление алкоголя (порции/день) | Целые значения от 0 до 5; без пропусков |
| SmokingStatus  | object | Статус курения | Бинарный: 'Yes' / 'No'; сбалансирован (515 / 485) |
| ExerciseFrequency  | int64 | Частота физических упражнений | дней в неделю (0–7); без пропусков |


## 3. Основные проблемы качества данных

| № | Тип проблемы | Описание | Потенциальный риск | Предлагаемое решение |
|---|--------------|----------|-------------------|---------------------|
| 1 | Пропуски | CaffeineConsumption: 256 пропусков (25,6%) | Смещение при анализе влияния кофеина на сон | Импутация медианой или удаление строк  |
| 2 | Формат данных | Bedtime, WakeupTime хранятся как строки, а не как время | Невозможно напрямую анализировать временные паттерны или вычислять продолжительность сна | Преобразовать в datetime, затем извлечь десятичное число |
| 3 | Отсутствие целевой переменной | Нет явного признака «нарушение сна» | Сложность формулировки задач классификации | Создать производную переменную, например: PoorSleep = SleepEfficiency < 0.75 |
| 4 | Семантическая неоднозначность | Не указаны единицы измерения (мг? чашки? порции?) | Затруднена интерпретация результатов для заказчика | Документировать предполагаемые единицы: кофеин — мг/день, алкоголь — стандартные порции/день |
| 5 | Возможные ошибки в данных | В SleepDuration встречаются значения < 4 часов (физиологически маловероятны) | Искажение выводов о паттернах сна | Проверить на соответствие разнице WakeupTime – Bedtime; при необходимости — корректировать или исключать |

## 4. План улучшения качества

**Приоритет 1 (критично):** Обработка пропусков в столбце CaffeineConsumption (256 значений, 25,6% от выборки) — импутация медианой или удаление строк.

**Приоритет 2:** Преобразование временных признаков Bedtime и WakeupTime из строкового формата в числовой для корректного расчёта продолжительности сна и анализа циркадных ритмов.

**Приоритет 3:** Проверка и нормализация суммы фаз сна (REMSleepPercentage + DeepSleepPercentage + LightSleepPercentage = 100%) и создание производной целевой переменной (например, PoorSleep = SleepEfficiency < 0.75) для последующего моделирования.

## 5. Краткое резюме

Образ жизни напрямую формирует структуру и качество сна: вредные привычки (курение, алкоголь, кофеин) ухудшают ключевые параметры сна, тогда как физическая активность — улучшает их. Также частые ночные пробуждения напрямую снижают эффективность сна — чем больше пробуждений за ночь, тем меньше времени человек проводит в непрерывном сне, что указывает на прерываемый, некачественный сон.
